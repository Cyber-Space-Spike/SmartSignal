O que é este protótipo

Versão web de validação do SmartSignal. Roda no navegador do seu Android com GPS real, usando os semáforos que você cadastrar. Serve para testar a experiência de direção antes de virar app baixável.

Regra inviolável: o app nunca controla o veículo — apenas sugere. Nunca sugere velocidade acima do limite.

Os dados ficam só no seu aparelho (armazenamento local). Nada é enviado para servidores.
